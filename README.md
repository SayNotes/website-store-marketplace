# 🛒 Website Store Marketplace

## 📌 Overview
Website Store Marketplace is a web-based platform designed for online buying and selling. It enables sellers to create stores, manage products, and process orders, while buyers can browse, compare, and purchase items seamlessly.

## 🚀 Key Features
- **User Authentication**: Registration, login, and account management.
- **Store Management**: Sellers can create stores, add products, and manage inventory.
- **Product Catalog**: Search, filter, and categorize products.
- **Shopping Cart**: Add, remove, and checkout products.
- **Payment Integration**: Support for payment gateways (e.g., Midtrans, PayPal).
- **Admin Dashboard**: Monitor transactions, users, and stores.

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
